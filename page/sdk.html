Datastar SDK

There are five types of server sent events in Datastar

  • Merge Fragment (our main workhorse)
  • Remove Fragment
  • Merge Signal (very useful if interacting with web-components)
  • Remove Signal
  • Execute Script (mostly for interacting with browser api's)

Datastar Events follow a specific format

retry: 1000
data: event datastar-merge-fragment
data: mergeMode append
data: selector #main
data: fragment <p>Hello World</p>

To help end user's, SDK's have been developped for various backend languages.

I maintain the typescript SDK, though it has not yet been published to npm.

I also maintain the SDK test suite, written in shell so all SDK's can share the same tests
